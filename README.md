## personal dotfiles

### Setup

Just follow [this](https://www.anand-iyer.com/blog/2018/a-simpler-way-to-manage-your-dotfiles.html) article

or install via the setup script:
* `wget https://github.com/TheWalkingLeek/dotfiles/blob/master/setup.sh`
* `./setup.sh`
